LinuxHostLoad工具需要系统安装sysstat包（提供sar、mpstat、pidstat、iostat等命令）
编译:

go build -o LinuxHostLoad_v1 LinuxHostLoad.go 

使用:
1.chmod 700 LinuxHostLoad_v1  
2.执行 ./LinuxHostLoad_v1
